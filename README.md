     Banking App
A simple Python command-line application that allows users to:

Create a bank account

Deposit funds

Withdraw funds

🚀 Getting Started
Requirements
Python 3.8+

pipenv

Installation & Run
bash
Copy
Edit
pipenv install
pipenv run python3 -m banking_app.main

📁 Features
Create a new account
Deposit money
Withdraw money (with balance check)
